Le dosseir "css" correspond au code css utilisé pour le site web

Le dossier "img" est le lieu de stockage des images utilisés pour le site web

le dossier "js" correspond au code javascript utilisé pour le site web

Pour lancer le site sans utiliser la base de données, glisser une des pages finissant par ".php" dans le navigateur

Pour lancer le site en utilisant la base de données, mettre le dossier "App_web" dans le dossier "www" de Wamp puis accéder a la page "localhost" depuis votre navigateur.





a_propos.php = Explication et origine de la création de ce projet

code_source.php = redirection vers la plateforme de stockage en ligne (accesible uniquement lorsque l'on est connecté a github)

impossible.php = lorsque un trajet est impossible redirection vers cette page

index.php = page principale pour effectuer des recherches de trajet

loading.php = page de vérification et de redirection pour le trajet

result.php = lorsque un trajet est possible rediriger vers cette page qui va afficher le résultat de la recherche

tabs.php = page qui regroupe les horaires des différentes ligne de bus
